# Document Registry

This registry lists the main documentation files that explain the public portfolio version of Local AI Workspace.

## Core documents

- `architecture.md` — system structure and module boundaries.
- `authentication_policy.md` — local login and session model.
- `audit_log_policy.md` — audit logging and redaction expectations.
- `memory_policy.md` — memory boundaries and privacy rules.
- `memory_governance_policy.md` — memory listing, export, and deletion behavior.
- `rag_source_policy.md` — source-aware retrieval expectations.
- `tool_permission_policy.md` — tool execution boundaries.
- `tool_risk_policy.md` — risk levels for tool actions.
- `ai_evaluation_policy.md` — deterministic AI behavior checks.
- `code_rewrite_protocol.md` — safe change and rollback protocol.

## Rule

Documentation should describe only features that are implemented or clearly marked as planned.

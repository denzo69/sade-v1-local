# Web Search Policy

Web search is used only when current or external facts are needed.

## Truth boundary

Search results are sources to inspect, not automatic proof. The assistant should show uncertainty when sources are weak, unavailable, or not yet reviewed.

## Rules

- Do not claim a web search was performed unless it actually ran.
- Do not invent source titles or URLs.
- Do not automatically write web results into memory.
- Keep source review separate from long-term memory storage.

## Portfolio value

This demonstrates source-aware AI behavior instead of unsupported model guessing.

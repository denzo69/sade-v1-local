# Operating Manual

This manual describes the intended operating model for Local AI Workspace.

## Normal use

1. Log in locally.
2. Use Chat as the primary workspace.
3. Add or review memory only when needed.
4. Add source documents when retrieval should be grounded.
5. Use advanced tools for review, debugging, and portfolio demonstration.

## Safety model

Memory writes, file changes, source review, and high-risk tool actions should remain explicit and auditable.

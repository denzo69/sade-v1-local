# Development Roadmap

Local AI Workspace is currently a portfolio-beta project.

## Current focus

1. Keep the public repository clean and understandable.
2. Maintain a working FastAPI application with browser UI.
3. Keep authentication, CSRF protection, audit logging, and memory governance testable.
4. Demonstrate retrieval, source quality checks, tool-risk policies, and AI evaluation entrypoints.
5. Keep screenshots, quickstart instructions, and documentation in English for a wider audience.

## Next useful improvements

- Add more end-to-end tests for the complete chat and memory workflow.
- Add a small demo video or animated GIF.
- Improve release notes and deployment documentation.
- Keep dependencies and GitHub Actions versions current.

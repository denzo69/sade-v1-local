# Project Memory Notes

This document describes the project-memory concept used by Local AI Workspace.

The system separates:

- user-facing chat history,
- durable project memories,
- source documents,
- audit logs,
- and technical debug traces.

Project memory is meant for useful continuity, not raw transcript storage. Personal or sensitive information should not be committed to the repository.

## Portfolio boundary

The public repository demonstrates the architecture and safeguards. It must not include private memory data from a real user environment.

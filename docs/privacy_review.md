# Privacy Review

This project is local-first, but a public portfolio repository still needs privacy hygiene.

## Repository checks

- Do not commit `.env` files.
- Do not commit local auth files.
- Do not commit sessions, backups, exports, uploads, vector databases, or private memories.
- Keep screenshots cropped to the app only.
- Keep documentation free of personal operational details.

## Current public posture

The repository should show architecture, safeguards, tests, and UI polish without publishing private user data.

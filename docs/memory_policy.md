# Memory Policy

Local AI Workspace separates durable memory from temporary chat context.

## What memory is for

- Project preferences.
- Explicitly saved corrections.
- Useful long-term context.
- Source summaries approved by the user.

## What memory is not for

- Passwords or tokens.
- Raw private chat dumps.
- Unreviewed web-search results.
- Sensitive personal details.

## Rule

When in doubt, do not store it automatically.
